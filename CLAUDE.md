# Cuckoo Clock — プロジェクト概要

## プロジェクトの目的

オルゴール（ミュージックボックス）用ディスクの STL ファイルを生成する Web アプリ。
3D プリントで製作したディスクを実際のオルゴール機構にセットして使用する。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | STL ジェネレーター + Three.js 3D プレビュー（唯一のファイル） |
| `forward.m4a` | ディスクを正方向に回転させたときの録音（5音サイクル） |
| `backward.m4a` | ディスクを逆方向に回転させたときの録音（5音サイクル） |
| `low.m4a` | φ30mm 単音ディスクの録音（全25セクター同一径） |
| `mid.m4a` | φ35mm 単音ディスクの録音（全25セクター同一径） |
| `high.m4a` | φ40mm 単音ディスクの録音（全25セクター同一径） |

## ディスク仕様

| パラメーター | 値 |
|---|---|
| セクター数 | 25 等分 |
| 外径パターン | φ40 → 37.5 → 35 → 32.5 → 30 mm × 5 回繰り返し |
| 厚さ | 1.8 mm |
| 中央穴 | 10.5 × 10.5 mm 正方形 |

## 音階解析結果（2026-05-15 時点）

### 前提
- **大きい径 = 高い音**（ユーザー確認済み）
- forward.m4a: φ40 → φ37.5 → φ35 → φ32.5 → φ30 の順（高 → 低）
- backward.m4a: φ30 → φ32.5 → φ35 → φ37.5 → φ40 の順（低 → 高）
- 5音 × 5サイクル = 25音（25セクターと一致）

### 5音サイクルディスク解析（forward.m4a / backward.m4a）

| 径 | 推定音名 | 観測周波数 | 備考 |
|---|---|---|---|
| φ40mm（最高音） | **A7** | ~3436Hz | A7(3520Hz)より約42¢低め |
| φ37.5mm | **F#7** | ~2980Hz | F#7(2960Hz)より約12¢高め、ほぼ正確 |
| φ35mm | **E7** | ~2708Hz | E7(2637Hz)とF7(2794Hz)の中間あたり |
| φ32.5mm | 不明 | 不安定 | サイクルごとにスペクトルが変動 |
| φ30mm（最低音） | 不明 | 不安定 | 同上 |

## 単音ディスク解析結果（2026-05-20 時点）

単音ディスク（全25セクター同一径）を用いた精密測定。

### φ30mm（low.m4a）

- 打鍵検出数: **28回**
- 主要ピーク: **2392Hz** → **D7 +31¢**（D7標準 = 2349Hz）
- 二次読み: 2449Hz ≈ D#7 -28¢
- **結論: D7 / D#7 付近（約2400〜2450Hz）**

### φ35mm（mid.m4a）

- 打鍵検出数: **36回**
- 主要クラスター: 3333〜3464Hz、中心 **3406Hz → G#7 +43¢**（G#7標準 = 3322Hz）
- 非常に安定した単峰性スペクトル
- **結論: G#7（Ab7）、約+40¢ シャープ**

### φ40mm（high.m4a）

- 打鍵検出数: **25回**
- G#7クラスター(3296〜3416Hz): φ35mm と共通 → **ボディ共鳴（胴鳴り）**
- φ40mm 固有ピーク（φ35mm との差スペクトルより）:
  - 3758Hz (Bb7 +13¢)、3659Hz (Bb7 -33¢)、930Hz (A#5)、853Hz (G#5)
- **結論: 真の音名は Bb7 / A#7（約3660〜3760Hz）**
  - G#7 ピークは φ35mm と共有するボディ共鳴であり、琴爪の固有音ではない

### 単音ディスクまとめ

| 径 | 実測音名 | 代表周波数 | 備考 |
|---|---|---|---|
| φ40mm | **Bb7（A#7）** | ~3709Hz | G#7ピークはボディ共鳴、差分で同定 |
| φ35mm | **G#7（Ab7）** | ~3406Hz | 非常に安定 |
| φ30mm | **D7〜D#7** | ~2420Hz | D7よりやや高め |

### 分析上の注意点
- 音の基音が 2500〜4000Hz と高いのは、ディスクが小径（最大φ40mm）のため琴爪が短く高周波になるため
- 単音ディスクでは同じ音が25回繰り返されるためボディ共鳴が強く出やすい
- φ40mm と φ35mm が同じ G#7 付近のピークを持つのは共通ボディ共鳴が原因
- 解析ツール: Python + soundfile + scipy（macOS `afconvert` で m4a → WAV 変換）
