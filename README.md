# 鳩時計ディスクジェネレーター

3Dプリンターで製作できるオルゴール（ミュージックボックス）用ディスクの STL ファイルを、ブラウザ上で生成するWebアプリです。

## 概要

オルゴール機構の「琴爪（きんそう）」は、ディスクの歯（突起）の高さに応じてどの音が鳴るかが決まります。このアプリでは 25 分割されたディスクの各セクターに音符または休符を割り当て、好みのメロディを奏でるディスクを設計・出力できます。

## 使い方

1. ブラウザで `index.html` を開く（ローカルファイルとして直接開くか、静的 HTTP サーバーで配信）
2. **固定パターン** または **メロディモード** を選択
3. メロディモードでは各ビートボタンをクリックして音を割り当て
4. 「▶ 試聴」でメロディを確認
5. 「STL をダウンロード」で 3D プリント用ファイルを取得
6. 3Dプリンターでプリントし、オルゴール機構にセット

### ボタン早見

| ボタン | 内容 |
|---|---|
| 固定パターン（元の設計） | φ40→37.5→35→32.5→30mm を 5 回繰り返す標準ディスク |
| メロディモード | 24 ビートを自由に設定できるメロディディスク |
| ★ キラキラ星 STL | キラキラ星のメロディディスクを直接ダウンロード |
| φ40mm（最高音）STL | 全セクター φ40mm の単音テスト用ディスク |
| φ35mm（中間音）STL | 全セクター φ35mm の単音テスト用ディスク |
| φ30mm（最低音）STL | 全セクター φ30mm の単音テスト用ディスク |

## ディスク仕様

| パラメーター | 値 |
|---|---|
| セクター数 | 25 等分 |
| 厚さ | 1.8 mm |
| 中央穴 | 10.5 × 10.5 mm 正方形 |
| 開始マーカー | 正方形穴の +X 方向の壁に V字溝（幅 2mm・深さ 1.5mm） |

中央の正方形穴は、オルゴール機構の回転軸に取り付けるためのものです。V字溝がある側がセクター 1（メロディの最初の音）の開始位置です。

## 歯の高さ（外径）と音階の関係

オルゴール機構では**外径が大きいほど高い音**が鳴ります。琴爪の長さが外径によって決まり、短い琴爪ほど高周波で振動するためです。

本アプリが対象とするオルゴール機構（最大外径 φ40mm）での実測値：

| 外径 | 半径（コード値） | 実測音名 | 代表周波数 | 備考 |
|---|---|---|---|---|
| **φ40mm** | 20 mm | **Bb7（A#7）** | ~3709 Hz | ボディ共鳴との差分で同定 |
| **φ37.5mm** | 18.75 mm | 参考値: F#7 | ~2980 Hz | 5音サイクルディスクより推定、精度低め |
| **φ35mm** | 17.5 mm | **G#7（Ab7）** | ~3406 Hz | 非常に安定した測定値 |
| **φ32.5mm** | 16.25 mm | 未同定 | 不安定 | サイクルごとにスペクトルが変動 |
| **φ30mm** | 15 mm | **D7〜D#7** | ~2420 Hz | D7 より約 30¢ 高め |

> **なぜこんなに高い音域なのか？**  
> ディスクの外径が最大 φ40mm と小径なため、琴爪が短くなり固有振動数が 2400〜3800 Hz（第 7 オクターブ）という高周波帯になります。

### 分析上の注意点

- φ40mm と φ35mm はどちらも G#7 付近（~3350 Hz）に強いピークを持ちますが、これは機構本体の**ボディ共鳴（胴鳴り）**です。φ40mm 固有の音は差分スペクトルで求めた Bb7（~3710 Hz）です。
- 単音ディスク（全 25 セクターが同じ外径）での測定値を優先しています。5音サイクルディスクでは機械ノイズの影響が大きくなります。
- 解析ツール: Python + soundfile + scipy（macOS `afconvert` で m4a → WAV 変換）

### メロディモードの音割り当て

アプリ内では次のように 5 段階の音を定義しています：

| ノート番号 | 外径 | 実測音 | プレビュー音（Web Audio） |
|---|---|---|---|
| 音1（高） | φ40mm | Bb7 | A4（440 Hz） |
| 音2 | φ37.5mm | 参考: F#7 | G4（392 Hz） |
| 音3 | φ35mm | G#7 | E4（330 Hz） |
| 音4 | φ32.5mm | 未同定 | D4（294 Hz） |
| 音5（低） | φ30mm | D7〜D#7 | C4（262 Hz） |
| 休 | — | 直前の音と同じ外径で出力 | 無音 |

> **休符について：** ディスク上では「休符」という概念はありません。エディター上で「休」を選んだビートは、STL 出力時に**直前のビートと同じ外径**として扱われます。これによりすべてのセクターが機構に確実にかかり、同じ音が 2 拍続く効果になります。

## キラキラ星について

デフォルトのメロディ「キラキラ星」は、5音ペンタトニックスケール（ファなし）に対応させています。ファの音はソ（φ37.5mm）で代替しています。

```
フレーズ1: ド ド ソ ソ ラ ラ ソ (ソ)
フレーズ2: ソ ソ ミ ミ レ レ ド (ド)
フレーズ3: ソ ソ ソ ソ ミ ミ レ (レ)
```

括弧内はエディター上では「休」ですが、ディスクでは直前の音が繰り返されます。

## ファイル構成

```
cuckoo-clock/
├── index.html   # HTML マークアップ
├── style.css    # スタイルシート
├── app.js       # Three.js プレビュー + STL 生成 + UI ロジック
└── README.md    # このファイル
```

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- ES Modules と importmap をサポートする環境
- Three.js は CDN（jsdelivr）から読み込み（インターネット接続が必要）

ローカルで動作させる場合は HTTP サーバーが必要です：

```bash
python3 -m http.server 8080
# または
npx serve .
```
